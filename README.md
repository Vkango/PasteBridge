# PasteBridge

一个 Windows 小工具, 用于在两个窗口之间快速复制并粘贴文本.

它适合这样的场景: 你需要频繁从一个窗口中选中文本, 并粘贴到另一个固定窗口的指定位置, 例如从网页、PDF、聊天窗口、代码窗口中摘录文本到笔记、表格或编辑器中. 

## 功能特性

- 绑定目标窗口 A
- 绑定源窗口 B
- 记录目标窗口 A 中的点击位置
- 从源窗口 B 复制当前选中文本
- 自动切换回目标窗口 A
- 点击之前记录的位置并粘贴文本
- 支持快速模式: 在源窗口 B 中拖拽选中文本后自动粘贴到目标窗口 A

## 快捷键

| 快捷键           | 功能                     |
| ---------------- | ------------------------ |
| `Ctrl + Alt + 1` | 绑定当前窗口为目标窗口 A |
| `Ctrl + Alt + 2` | 绑定当前窗口为源窗口 B   |
| `Ctrl + Alt + V` | 手动执行一次复制并粘贴   |
| `Ctrl + Alt + T` | 开启或关闭快速模式       |

## 使用方法

1. 启动 PasteBridge. 
2. 切换到需要粘贴文本的窗口, 按 `Ctrl + Alt + 1`, 将其绑定为目标窗口 A. 
3. 在目标窗口 A 中点击一次需要粘贴的位置, 程序会记录该位置. 
4. 切换到文本来源窗口, 按 `Ctrl + Alt + 2`, 将其绑定为源窗口 B. 
5. 在源窗口 B 中选中文本. 
6. 按 `Ctrl + Alt + V`, 程序会将选中的文本复制并粘贴到目标窗口 A 的记录位置. 
7. 也可以按 `Ctrl + Alt + T` 开启快速模式. 开启后, 在源窗口 B 中拖拽选中文本, 程序会自动完成复制和粘贴. 

## 注意事项

- PasteBridge 会使用系统剪贴板完成复制和粘贴, 所以在这期间请不要往剪贴板里放重要内容.
- 快速模式依赖鼠标拖拽行为判断是否完成文本选择.
- 如果目标窗口或源窗口关闭, 需要重新绑定.
- 某些管理员权限窗口、全屏程序或安全软件窗口可能无法正常激活或粘贴.

- 目前仅支持文本内容.
- 剪贴板内容可能会被工具覆盖.
- 目标窗口位置基于窗口 client 坐标记录, 窗口布局变化后可能需要重新点击记录位置.
- 对浏览器、PDF 阅读器、Electron 应用等复杂窗口, 快速模式可能受应用自身选区更新速度影响.
